## Installation
To get started with these Zsh scripts, ensure you have Zsh installed on your system. You can install Zsh using the following command:

```bash
sudo apt install zsh  # For Ubuntu
brew install zsh      # For MacOS
```

Clone this repository to your local machine:

```bash
git clone https://github.com/judempoyo/devScripts.git
cd devScripts
```

## Usage
After cloning the repository, you can start using the scripts immediately. To execute any script, use the following command:

```bash
sh script_name.zsh
```

Make sure you have the necessary permissions to run the scripts:

```bash
chmod +x script_name.zsh
```
